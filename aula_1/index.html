Tags HTML

Introdução

Bem-vindo a esta página que explora algumas tags HTML e sua funcionalidade.

Títulos

Os títulos são usados para destacar informações importantes em uma página.

<h1> até <h6>

Essas tags são usadas para definir cabeçalhos em um documento HTML, do mais importante (h1) ao menos importante (h6).

Destacar Texto

Essas tags são usadas para enfatizar ou destacar texto em uma página.

<mark>

Marca o texto com uma cor de fundo para destacá-lo. Exemplo: Texto destacado

<strong>

Indica que o texto é semanticamente importante e é exibido em negrito. Exemplo: Texto em negrito

<i>

Usado para enfatizar o texto, geralmente exibido em itálico. Exemplo: Texto em itálico

<u>

Sublinha o texto para destacá-lo. Exemplo: Texto sublinhado

Listas

Listas são usadas para agrupar itens relacionados em uma página.

<ol>

Cria uma lista ordenada, onde cada item é marcado com um número sequencial.

  1. Item 1
  2. Item 2
  3. Item 3

<ul>

Cria uma lista não ordenada, onde cada item é marcado com um marcador.

  • Item 1
  • Item 2
  • Item 3

<li>

Define um item em uma lista.

Links

Links são usados para conectar páginas ou recursos relacionados.

<a>

Cria um link para outra página da web, arquivo ou seção na mesma página.

Exemplo de link: Visite o exemplo.com

Linha Horizontal

A tag hr cria uma linha horizontal para separar seções de conteúdo.

<hr>

Cria uma linha horizontal.

Texto Sobrescrito e Subscrito

Essas tags são usadas para exibir texto sobrescrito ou subscrito em uma página.

<sup>

Exibe o texto como sobrescrito. Exemplo: X2

<sub>

Exibe o texto como subscrito. Exemplo: H2O
